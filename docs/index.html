strnow

Returns the current datetime as a string.
That's it. We promise.

Live demo

Watch it do its one job.

No config. No options. No drama. Just the time.

node repl
// one import to rule them all
import strnow from 'strnow';
 
// the only method. yes, just the one.
console.log( strnow.get() );

Installation

One line. Two options.

npm or yarn — your project, your call.

npm
$ npm install strnow
yarn
$ yarn add strnow

Usage

Three flavors. Same result.

ESM, CommonJS or TypeScript — strnow doesn't judge you.

ESM — Recommended 
import strnow from 'strnow';

console.log( strnow.get() );
// 2026-04-05 14:30:25

// Named import also works
import { get } from 'strnow';
console.log( get() );
CommonJS 
const strnow = require('strnow');

console.log( strnow.get() );
// 2026-04-05 14:30:25
TypeScript — types included 
import strnow from 'strnow';

const now: string = strnow.get();
console.log( now );
// 2026-04-05 14:30:25

// Named import also works
import { get } from 'strnow';
